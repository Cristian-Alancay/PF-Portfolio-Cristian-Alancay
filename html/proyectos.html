Car Easy

El proyecto "Car Easy" se centró en el desarrollo de una plataforma web para la gestión de alquiler de vehículos. La aplicación permite a los usuarios buscar, reservar y gestionar alquileres de coches de manera eficiente. Utilicé tecnologías modernas para crear una interfaz intuitiva y un backend robusto, asegurando una experiencia de usuario óptima y un rendimiento confiable.
